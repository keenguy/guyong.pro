Git branching

pro git 该文介绍了Git中commit和branch的工作原理。

git merge 是将两个branch merge起来。假设两个branch对应的commit分别为c1,c2. 其工作原理是：找到c1和c2的“最好的”公共祖先c3, 然后根据c1,c2,c3生成一个新的commit c4, c4中同时包括c1, c2对c3的”更改”。若两者“更改”导致的结果不同则会发生冲突(conflicts).
